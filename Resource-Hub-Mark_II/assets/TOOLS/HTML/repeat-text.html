Text repeater tool What is a text repeater?
With this tool, you can create multiple copies of the given text. You can set the number of repetitions in the options and adjust the delimiter for the output copies. You can also enter a decimal number in the repetition field. In this case, the integer part of the value is responsible for the number of full copies of the text and the fractional part adds a partial copy of the input text. Textabulous!
