Text printf tool What is a text printf?
With this tool, you can format text according to the format specified to printf function. As printf and sprintf functions share the same format and logic, we actually use sprintf function internally to perform formatting. You can specify the format in the options above. The format can contain one or more specifiers that consist of a percentage character followed by the format code and can include optional flags. Flags are width, precision, and length. You can find documentation of all the format codes and flags in "sprintf" function's (or "printf" function's) manual page. Textabulous!

Right Align Flower Names
In this example, we format the input text, which is a list of flower names, using the "%15s" format. The % character indicates the beginning of a specifier, number 15 specifies the length of the line, the symbol s denotes text type. When this format is applied to all text lines, spaces get added on the left so that every line was 15 characters in length.
