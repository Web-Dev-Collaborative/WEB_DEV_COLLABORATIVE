Text escaper tool What is a text escaper?
With this tool, you can slash-escape all special symbols in the given text. It has the same behavior as PHP"s addslashes() function. It adds a backslash before all double and single quotation marks, converts tabs to \t, converts newlines to \n, and each backslash gets replaced with two backslashes. You can now safely use this escaped text in the command line or place it between a new pair of quotes. Textabulous!
Text escaper examples Click to use
James Bond Quote
In this example, we slash-escape the farmous James Bond quote. As you can see, backslashes get added before before single and double quotation marks.
"The name's Bond. James Bond."
\"The name\'s Bond. James Bond.\"
Escape a Multi-line Dialogue
In this example, we escape a dialogue between Andy and Red from the movie Shawshank Redemption. Each newline is replaced with the \n character, double quote is replaced by \".
Andy: "You know what the Mexicans say about the Pacific?" Red: "No." Andy: "They say it has no memory. That's where I want to live the rest of my life. A warm place with no memory."
Andy: \"You know what the Mexicans say about the Pacific?\"\nRed: \"No.\"\nAndy: \"They say it has no memory. That\'s where I want to live the rest of my life. A warm place with no memory.\"
